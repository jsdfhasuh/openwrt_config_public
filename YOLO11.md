# YOLO11 信息 / YOLO11 Information

## YOLO11是哪个公司的？/ Which company is YOLO11 from?

**YOLO11 是由 Ultralytics 公司开发和维护的。**

**YOLO11 is developed and maintained by Ultralytics.**

## 关于 Ultralytics / About Ultralytics

Ultralytics 是一家专注于计算机视觉和深度学习的公司，自 YOLOv5 以来一直在开发和维护 YOLO 系列模型。

Ultralytics is a company focused on computer vision and deep learning that has been developing and maintaining the YOLO series models since YOLOv5.

## 关于 YOLO11 / About YOLO11

YOLO11 是 YOLO (You Only Look Once) 系列的最新版本，是一个先进的目标检测和图像分割模型。

YOLO11 is the latest version of the YOLO (You Only Look Once) series, an advanced object detection and image segmentation model.

### 主要特点 / Key Features

- 实时目标检测 / Real-time object detection
- 高精度和高性能 / High accuracy and performance
- 支持多种任务：检测、分割、分类、姿态估计等 / Supports multiple tasks: detection, segmentation, classification, pose estimation, etc.
- 易于使用和部署 / Easy to use and deploy
